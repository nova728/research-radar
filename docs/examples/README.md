## 如何生成 / How to Generate

1. 启动 ResearchRadar（`bash run.sh`）
2. 确保已抓取论文（趋势总览 → 立即抓取）
3. 前往**议题地图**页面，选择目标周
4. 点击绿色 **「导出 精选报告」** → 下载 `research-digest-YYYY-WNN.md`（含 LLM 逐篇解读）
5. 点击灰色 **「导出 MD」** → 下载论文列表 `research-radar-papers.md`

> 导出的 `.md` 文件可直接导入 **Obsidian、Notion、Typora** 等笔记工具。
